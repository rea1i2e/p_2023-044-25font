フォントファミリーについての検証

bodyタグに、以下の指定 

	body {
		font-family: 'Roboto','Noto Sans JP', sans-serif;
	}

英字フォントには、'Roboto'が反映される。

This font is a paragraph in engllish.

日本語文字列には、'Noto Sans JP'が反映される。

日本語の文字列。テキスト。テキスト。テキスト。

確認は検証ツールのElements>Consoleから

どのフォントが適用されているかは、Elements>Consoleから確認できる。
画面一番下の、Rendered Fontsに、実際に適用されているフォントファミリーが表示される。
日本語＋英語の文章の場合には、複数表示される。
